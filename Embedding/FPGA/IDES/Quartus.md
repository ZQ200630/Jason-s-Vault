# 可用IP

### Arthmetic（Basic Functions → Arthmetic）

- ALTERA_CORDIC 计算正弦余弦对应角
- ALTFP_ADD_SUB 加浮点减法器
- ALTFP_ATAN 浮点反正切
- ALTFP_COMPARE 浮点比较
- ALTFP_DIV 浮点除法
- ALTFP_SINCOS 浮点正弦余弦
- ALTFP_SQRT 浮点开根号
- ALTMEMMULT 数量乘法
- ALTMULT_ACCUM 乘加器
- ALTMULT_ADD 两路乘法后结果相加
- ALTMULT_COMPLEX 复数乘法
- ALTSQRT 带余开方
- FP_FUNCTIONS Intel FPGA IP 函数IP大全
- LPM_ADD_SUB 加减法
- LPM_COMPARE 比较
- LPM_COUNTER 计数器
- LPM_DIVIDE 除法
- LPM_MULT 乘法
- PARALLEL_ADD 多路加法器

## Logic(Basic Functions → Miscellaneous)

- ALTECC  编码
- LPM_DECODE  解码
- LPM_CLSHIFT 组合逻辑移位器
- LPM_MUX 多路选择器
- LPM_SHIFTREG 移位寄存器
- LPM_CONSTANT 把参数转化为常数，有点复杂
